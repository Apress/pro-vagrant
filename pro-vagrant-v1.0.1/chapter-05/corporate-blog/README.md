Corporate Blog
==============

To run the blog execute the following commands:

    # Host OS
    $ git clone https://github.com/pro-vagrant/corporate-blog.git
    $ cd corporate-blog
    $ vagrant up
    $ vagrant ssh
    $ cd /vagrant
    $ jekyll serve -H 0.0.0.0 --detach

Open your web browser and visit:

    http://127.0.0.1:8080
